# Local Source Library (originals)

Downloaded originals of the public, open, and openly-licensed documents that the wiki cites — kept alongside the `.md` wiki so the project has authoritative primary copies, not just summaries.

## Policy

- **Store only** public-domain (US federal government: FDA, eCFR/CFR, NIH), openly-licensed, or explicitly redistributable materials, plus openly-published academic/CRO templates whose terms permit reuse.
- **Do not store** gated, paywalled, or proprietary materials (e.g., MedDRA, subscription journal PDFs, consortium members-only templates). For those, keep the citation and URL only, and record the license status.
- Every stored file is recorded in `MANIFEST.md` with: filename, title, authority/source, URL, retrieval date, and license/terms.
- Full-text peer-reviewed publications are stored **only** where the license permits (open-access / CC / author-posted). Otherwise: citation + DOI/PMID only.

## Layout

| Folder | Contents |
|--------|----------|
| `cfr/` | 21 CFR Parts 11, 50, 54, 56, 312 (and 45 CFR 46, 42 CFR Part 11) — eCFR/GovInfo copies |
| `ich/` | ICH guidelines (E6(R3), E8(R1), E9/E9(R1), E2A/E2B(R3)/E2D/E2F, E3, M11) |
| `fda-guidances/` | FDA guidance PDFs (DCT 2024, AI credibility 2025 draft, eConsent, sponsor-investigator IND, investigator responsibilities, DMC, RBM, expanded access) |
| `fda-forms/` | FDA forms (1571, 1572, 3454, 3455, 3500A, 3926, 3674) |
| `institutional-templates/` | Openly-published academic/CRO templates (e.g., NIH-FDA protocol template, TransCelerate CPT where license permits, CTSA toolkits) with license notes |
| `publications/` | Open-access publications only (CC/author-posted); otherwise citation-only in MANIFEST |
| `other/` | Anything not fitting the above, with provenance |

See `MANIFEST.md` for the itemized index. License provenance for templates is also tracked in `../references/external-templates-and-licenses.md`.
